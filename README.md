# Node Express MySQL Restful 

## Install Dependencies 
```
npm intall
```

## Development Server 
```
npm run dev
```

http://localhost:3030/

## Product Server 
```
npm run start:prod
```

```
http://localhost:3030/users
http://localhost:3030/user/user_id
```

## POST
```
http://localhost:3030/signup
http://localhost:3030/messages
http://localhost:3030/sendMessage
```
